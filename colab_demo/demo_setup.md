# Running the Colab Demo

1. Create a free ngrok account: https://ngrok.com/
2. Copy your Auth Token from the dashboard.
3. Open `demo_code.ipynb`.
4. Paste the token at the end section of the code updating the place holder: `PASTE_YOUR_NGROK_API_KEY_HERE`.
5. Run the code
6. Press `Ctrl + left mouse click` on the first public link
